## Practice Task

Choose the best standard container for five access/update scenarios.

## Target Skill

Container fit.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Define scenarios: indexed access, queue, lookup, ordered traversal, stack.
2. Pick a container.
3. Justify each choice.
4. Name a tradeoff.

## Success Check

- Each choice maps to operations.
- Tradeoffs are explicit.

## Common Failures

- Choosing vector for everything.
- Ignoring lookup/update patterns.

## Notes

Keep the drill short, repeatable, and measurable.

## Modernization

- Use current compiler/toolchain and modern C++ defaults.
