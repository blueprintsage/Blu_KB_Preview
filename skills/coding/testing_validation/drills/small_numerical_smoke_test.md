## Practice Task

Create a tiny expected-output test for a numerical transform.

## Target Skill

Numerical verification.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Choose two input records.
2. Calculate expected transformed values.
3. Run transform.
4. Compare output within tolerance.

## Success Check

- Expected values are documented.
- Tolerance is stated.

## Common Failures

- Exact compare of floating text.
- No expected result.

## Notes

Keep the drill short, repeatable, and measurable.

## Modernization

- Use current compiler/toolchain and modern C++ defaults.
