## Objective
Reproduce the bug, isolate the input, inspect state, test one change, and confirm output.

## Steps / Flow
1. Define the visible goal.
2. Identify inputs and outputs.
3. Build the smallest working version.
4. Add one behavior at a time.
5. Verify with console, Serial, or visible output.
6. Refactor names and structure.
