# Program Options Matrix Drill

## Purpose

Practice separating flags, defaults, config-file values, and validation rules.

## Use When

A learner needs hands-on practice applying the corresponding Boost/modern C++ decision.

## Pattern

1. List five options for a small CLI tool.
2. Mark required vs optional.
3. Assign command-line and config-file sources.
4. Define defaults.
5. Write validation rules before execution.

## Modern C++ Note

Use modern standard facilities where they express the same idea more directly; keep Boost forms when they teach portability or remain the strongest tool.
