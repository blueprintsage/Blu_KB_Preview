## Practice Task

Print the same numeric values in default, fixed, scientific, and compact formats.

## Target Skill

Output formatting control.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Choose three doubles.
2. Print with default formatting.
3. Print with fixed precision.
4. Print with scientific precision.
5. Explain the use case for each.

## Success Check

- Output differences are visible.
- Precision is intentionally selected.

## Common Failures

- Leaving stream state changed accidentally.
- Confusing display precision with stored value.

## Notes

Keep the drill short, repeatable, and measurable.

- Use current compiler/toolchain and modern C++ defaults.
