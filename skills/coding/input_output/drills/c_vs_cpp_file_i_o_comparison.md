## Practice Task

Implement the same file transform once with C++ streams and once with C FILE* APIs.

## Target Skill

API contrast and transfer.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Write C++ stream version.
2. Write C stdio version.
3. Identify equivalent open/read/write/close operations.
4. Summarize one advantage/risk of each.

## Success Check

- Both versions produce same output.
- Differences are explained, not memorized.

## Common Failures

- Mixing APIs without understanding.
- Copying syntax only.

## Notes

Keep the drill short, repeatable, and measurable.
