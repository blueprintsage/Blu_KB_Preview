## Objective

Turn raw memory or device access into a safer C++ boundary.

## Steps / Flow

1. Locate raw address/cast/API contact.
2. Create a narrow class interface.
3. Hide dangerous operations.
4. Document timing and failure assumptions.
5. Provide a fake or test double for tests.
