## Practice Task

Trace how assigning through *p changes the original variable.

## Target Skill

Pointer mechanics.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Create int q and int* p=&q.
2. Predict q after *p assignment.
3. Run or reason through it.
4. Explain value vs address.

## Success Check

- Correct before/after values.
- Clear explanation of dereference.

## Common Failures

- Confusing p with *p.
- Forgetting & produces address.

## Notes

Keep the drill short, repeatable, and measurable.

- Use current compiler/toolchain and modern C++ defaults.
