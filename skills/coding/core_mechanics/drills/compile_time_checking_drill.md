## Practice Task
Practice applying: compile-time checking drill.

## Target Skill
Recognize and apply the C++ concept in small code or design examples.

## Setup
Use a small self-contained C++ example or pseudocode sketch.

## Instructions
1. Identify the relevant C++ concept.
2. Write the simplest correct version.
3. State the invariant, ownership, or interface rule involved.
4. Rewrite once using a more modern or more explicit form.

## Success Check
- The result compiles conceptually.
- Ownership and type intent are clear.
- No unnecessary abstraction was added.

## Common Failures
- Copying syntax without understanding the design rule.
- Using legacy style when a safer standard facility exists.
- Overbuilding the example.

## Variants

### Variant: Review Mode
Repeat the task by reviewing an existing snippet and marking the exact line where the concept is violated.
