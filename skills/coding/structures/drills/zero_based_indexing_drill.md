## Practice Task

Translate one small Fortran-style loop to C++ zero-based indexing.

## Target Skill

Index discipline.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Start with mathematical 1..n notation.
2. Write C++ 0..n-1 loop.
3. Map displayed labels separately if needed.
4. Check boundary cases.

## Success Check

- No off-by-one error.
- Loop covers all intended elements once.

## Common Failures

- Using <= n.
- Mixing display index with storage index.

## Notes

Keep the drill short, repeatable, and measurable.

## Modernization

- Use current compiler/toolchain and modern C++ defaults.
