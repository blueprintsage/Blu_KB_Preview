## Objective
Practice list insertion and deletion by naming every pointer change before coding it.

## Steps / Flow
1. Draw the nodes involved.
2. Name the predecessor, target, and successor.
3. Write the relinking order in plain language.
4. Handle head and empty-list cases separately.
5. Only then translate the relinking into code.
