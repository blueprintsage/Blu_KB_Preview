## Practice Task
Rewrite five beginner exercises so each teaches only one new concept.

## Target Skill
Beginner coding execution and recognition.

## Setup
Use a small local example; keep the exercise repeatable.

## Instructions
1. Read the prompt.
2. Apply the named concept.
3. Check the visible output or trace.

## Success Check
- Result matches expected behavior.
- Learner can explain the rule used.

## Common Failures
- Changing too many things at once.
- Confusing syntax errors with logic errors.

## Modernization
Use current JavaScript or current Arduino IDE conventions rather than preserving old source-era tooling.
