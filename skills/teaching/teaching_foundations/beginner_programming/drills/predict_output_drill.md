## Objective
Train students to execute code mentally before relying on a compiler.

## Steps / Flow
1. Choose a short program with one new concept.
2. Write the current value of each relevant variable after each statement.
3. Predict the exact output.
4. Run the program.
5. Compare prediction to actual output and identify the first divergence.

## Notes
The textbook uses predict-the-output exercise families throughout the course. PASS keeps the exercise format, not the original problems.
