## Pattern Rule
**IF** teaching beginners a programming concept  
**THEN** start with a visible concrete result  
**ELSE** abstraction arrives without anchor

## Do
- Keep the primary skill goal intact.
- Use the modernized form in new work.

## Don't
- Preserve outdated syntax or tooling when a current form is safer.
