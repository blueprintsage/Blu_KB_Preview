## Objective

Find hidden assumptions by writing targeted tests.

## Steps / Flow

1. List three assumptions in the code.
2. Write one test per assumption.
3. Break one assumption intentionally.
4. Change the code or precondition.
5. Add the regression check.

## Success Check
