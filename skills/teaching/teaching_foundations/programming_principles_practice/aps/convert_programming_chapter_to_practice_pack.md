## Objective

Turn a chapter into a reusable drill/exercise/project/review pack.

## Steps / Flow

1. Extract the chapter's main skill.
2. Write one mechanical drill.
3. Write three exercises with increasing variation.
4. Add one integration task.
5. Add a success rubric and common failure list.

## Success Check

- The workflow ends with a runnable artifact, test plan, or teaching pack.
- Dated C++ is modernized unless intentionally marked mechanics-only.
