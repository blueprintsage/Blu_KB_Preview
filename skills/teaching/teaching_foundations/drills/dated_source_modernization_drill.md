## Practice Task

Given a dated snippet, separate timeless lesson from outdated execution.

## Target Skill

Source literacy.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Identify the lesson.
2. Mark stale APIs/syntax/tools.
3. Write a modern equivalent.
4. Explain what stayed the same.

## Success Check

- Concept preserved.
- Execution updated.

## Common Failures

- Rejecting useful concepts.
- Teaching old API as current.

## Notes

Keep the drill short, repeatable, and measurable.

## Modernization

- Use current compiler/toolchain and modern C++ defaults.
