## Practice Task

Take one C++ example and create a goal-structure-code-dissection teaching sequence.

## Target Skill

Teaching executable examples.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. State the learning goal.
2. Show expected run/output.
3. Show program structure.
4. Reveal code.
5. Dissect line groups.

## Success Check

- Learner sees purpose before syntax.
- Dissection follows code, not replaces it.

## Common Failures

- Starting with every detail.
- No runnable anchor.

## Notes

Keep the drill short, repeatable, and measurable.

## Modernization

- Use current compiler/toolchain and modern C++ defaults.
