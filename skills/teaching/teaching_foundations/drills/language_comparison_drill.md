## Practice Task

Use C, C++, and Python versions of a tiny task to teach abstraction levels.

## Target Skill

Teaching transfer.

## Setup

Use a small local C++ project and one source file.

## Instructions

1. Show all versions.
2. Identify shared goal.
3. Compare visible complexity.
4. Name what each abstraction hides.

## Success Check

- Learner can explain abstraction tradeoff.
- Comparison does not become language ranking.

## Common Failures

- Mocking low-level code only.
- Losing the shared problem.

## Notes

Keep the drill short, repeatable, and measurable.

## Modernization

- Use current compiler/toolchain and modern C++ defaults.
