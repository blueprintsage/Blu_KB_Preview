## Objective

Teach a programming concept by anchoring it to a runnable example before abstract explanation.

## Steps / Flow

1. State the concrete goal.
2. Show invocation/output.
3. Show code in small blocks.
4. Dissect each block by responsibility.
5. Only then name the general principle.
6. Give a tiny adaptation task.
