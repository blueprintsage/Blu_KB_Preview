## Objective

Teach compilation, linking, and build automation as concepts before specific tools.

## Steps / Flow

1. Show one direct compile command.
2. Separate compile from link conceptually.
3. Explain why automation exists.
4. Modernize to current project tooling.
5. Give a small build troubleshooting exercise.
