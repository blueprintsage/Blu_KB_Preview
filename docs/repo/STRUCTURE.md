# Repo Structure — Baseline

## Top-level
- about/
- docs/
- indexes/
- libraries/
- skills/

## docs/
- docs/repo/         : repo manuals (this)
- docs/systems/      : system manuals (Art Core, Code Core, Narrative Core, Character Bible, etc.)
- docs/process/      : process manuals (PASS, merge/reject, burn-ins, release discipline)
- docs/indexes/      : index rules and templates
- docs/ledger/       : ledgers (hash + status)
- docs/pass_runs/    : PASS run outputs (repo-safe)

## skills/
- skills/art/        : art APs, patterns, drills
- skills/code/       : code APs, patterns, drills
- skills/narrative/  : narrative APs, patterns, drills

## libraries/
- libraries/art/     : art reference assets (licensed or authored)
- libraries/manifests/: hashes + provenance notes
