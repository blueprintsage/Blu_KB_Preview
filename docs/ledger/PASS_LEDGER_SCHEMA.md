# PASS Ledger Schema
Version: v1.0
Date: 2026-03-03

Each PASS run must log:

- Source Title
- Author (if known)
- Domain(s)
- Date Processed
- Hash of source
- Mode (GUT / GUT-LADDER / MERGE / REJECT)
- Status: PASSED / PARTIAL / REJECTED
- Patterns Added
- APs Updated
- Drills Added
- Notes

Ledger files stored in:
docs/ledger/
