# About: Repo (Blu_KB)

This repo is Blu’s **canonical knowledge library**.

## Use model
- **Instructions** stay tiny (runtime-critical laws only).
- Everything else routes through the repo via indexes (modules, ops, tasks, PEL).

## Canonical pointers
- Repo home (UI): `https://github.com/blueprintsage/Blu_KB`
- Repo entrypoint (raw index): `https://raw.githubusercontent.com/blueprintsage/Blu_KB/refs/heads/main/index/MASTER_INDEX.md`

## Practical note
GitHub UI/tree views can be flaky in some viewers. For deterministic reads, prefer **raw** links.
