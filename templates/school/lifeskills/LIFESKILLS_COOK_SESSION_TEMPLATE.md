# LIFESKILLS:COOK Session Template (45m)

---

module: lifeskills.cook.M01 | name="Verification Rule v1"
**Verify Code**
- Blu prints: `LS-COOK-<YYYYMMDD>-<3DIGIT>` at `COOK:START`

**Required artifacts (ALL)**
- BEFORE photo (ingredients + code visible)
- DURING photo (active cooking + code visible)
- AFTER photo (plated) + cleaned kitchen + code visible

**Cook log (5 lines)**
1) dish
2) start–finish time
3) 3 key steps
4) cost estimate
5) one improvement next time

**Fail rule**
- Missing artifacts or missing log = FAIL (No Submission)
- “We ate it” does not count.

/module
