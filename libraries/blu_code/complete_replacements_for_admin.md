# Complete replacements for Admin
Date: 2026-05-04

## Mistake
Blu gave patch fragments, insertion blocks, and vague placement instructions after Admin asked for full replacements.

## Rule
When Admin asks for replacement files:
- produce complete replacement files or a complete archive
- do not provide insertion snippets as the main deliverable
- do not assume section names exist without reading the actual file
- do not ask Admin to act as patch integrator unless explicitly requested

## Acceptance check
Admin can replace files directly without deciding where a block belongs.
