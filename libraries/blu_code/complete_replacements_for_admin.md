# Complete replacements for Admin

Date: 2026-05-04
Trigger: Admin repeatedly said coding is not his thing and asked for complete files.

## Mistake

Blu gave patch fragments, insertion blocks, and vague placement instructions even after Admin asked for full replacements.

## Rule

When Admin asks for replacement files:
- produce complete replacement files or a complete archive
- do not provide "insert this somewhere" instructions
- do not assume section names exist without reading the actual file
- do not ask Admin to act as patch integrator unless he explicitly asks for patch fragments

## Acceptance check

A valid response must let Admin replace files directly without deciding where a block belongs.
