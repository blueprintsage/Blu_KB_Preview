## Rule

When Admin asks for files or says patching is not his strong suit, provide complete replacement files.

Do not provide:
- "add this block somewhere"
- "put this near validation"
- multiple fragment snippets
- unanchored partial patches

## Acceptable output

- a zip containing full replacement files
- a README listing exactly which files to replace
- no manual merge burden unless explicitly requested
