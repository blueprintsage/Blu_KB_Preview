## Mini-Lesson 1 — A life skill is not "knowing about it"

rule:
- Knowing what to do is not the same as being able to do it.

examples:
- "I know budgets matter" is not a budget.
- "I know calendars help" is not a working weekly plan.
- "I know emails should be professional" is not a clean, usable email draft.

better measure:
- Can you do the task?
- Can you show the result?
- Can you repeat it later?

## Mini-Lesson 2 — The practical sequence

sequence:
1. Identify the task
2. Gather what you need
3. Do the task in order
4. Check the result
5. Save proof / record completion
6. Reflect on one improvement

example:
task: plan the week
- identify appointments, due dates, work, chores, and priorities
- place them into a realistic weekly layout
- leave breathing room
- check for conflicts
- save the plan
- note one place the plan could fail

## Mini-Lesson 3 — Communication should reduce friction

good communication:
- clear subject
- clear ask
- correct tone
- enough detail
- not too much detail
- no rambling

bad communication:
- vague message
- missing ask
- messy tone
- no context
- late response with no explanation

## Mini-Lesson 4 — A system beats motivation

rule:
- A weak system makes every day harder.
- A simple reliable system beats waiting until you "feel like it."

examples:
- a calendar beats memory
- a checklist beats vague intention
- a saved template beats starting from zero every time
- one folder for important items beats hunting everywhere
