## Day 132 Workbook Page

day_number: 132
lesson_title: Mixed Review II — quadratics, rational expressions, radicals
tutoring_mode: CHECK_ONLY
evidence_required:
- visible work for all six problems
- restrictions shown for rational problems
- check shown for radical equation

instructions:
- Complete all six problems.
- Show the main steps, not just the final answer.
- If you get stuck, mark the step where you got stuck instead of skipping the problem silently.

problems:

1. Solve:
   x^2 - 7x + 10 = 0

workspace:
- factor or use another valid method
- list both solutions

2. Solve:
   x^2 + 6x - 7 = 0

workspace:
- choose a valid solve path
- simplify fully

3. Simplify and list restrictions:
   (x^2 - 4)/(x^2 - x - 2)

workspace:
- factor numerator
- factor denominator
- list all original restrictions
- simplify only after restrictions are recorded

4. Solve and check restrictions:
   4/x = 12/(x+2)

workspace:
- state restrictions first
- solve
- reject invalid values if needed

5. Simplify:
   81^(3/4)

workspace:
- show the root/power logic

6. Solve and check:
   sqrt(x+1)=5

workspace:
- solve
- check in the original equation
