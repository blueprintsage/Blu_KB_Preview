# Naming conventions (v1)

- lowercase
- underscores
- no spaces
- stable suffixes: _front, _back, _3q, _profile_left, _profile_right

Recommended compiled sheet names:
- canon_sheet_full.png
- body_sheet_starter_front_back_3q.png
- headpack_sheet_starter_turns_expressions.png
- hair_sheet_starter_angles_highlight_map.png
- clothing_sheet_starter_seams_emblems.png
- equipment_sheet_starter_rig_weapons.png
